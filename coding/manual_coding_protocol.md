# Manual Coding Protocol: Justification Types

## HOW TO CODE

code_ACCT: Enter 1 if accountability justification present, leave blank if not
code_EFFI: Enter 1 if efficiency justification present, leave blank if not
code_OTHER: If neither fits, briefly note what the justification is

Codes are NOT mutually exclusive - you can mark 1 in both ACCT and EFFI.

---

## ACCT (Accountability-Oriented)

References to responsibility, liability, professional obligations, being able to explain/defend work, attribution, trust, credibility.

Indicators:
- "I need to be able to explain/defend this"
- "I would be responsible/liable if..."
- "I should be accountable for..."
- "My clients/students/colleagues trust me to..."
- "It's my professional responsibility"
- "I need to understand what I'm submitting"
- "If something goes wrong, it's on me"
- References to professional codes, ethics, standards
- Concerns about attribution, credit, authorship

NOT ACCT:
- General quality concerns without responsibility framing
- Personal preference without professional stakes

---

## EFFI (Efficiency-Oriented)

References to time-saving, productivity, output speed, workload management, keeping up, competitive pressure, getting more done.

Indicators:
- "Saves time"
- "Faster" / "quicker"
- "More efficient" / "productive"
- "Gets more done"
- "Keeps up with workload"
- "Competitive pressure"
- "Everyone else is doing it"
- "Why not use it?"
- "Helps me" (instrumental framing)
- "Polish" / "improve" for practical reasons

NOT EFFI:
- Quality improvements framed as authentic expression
- Time references without productivity framing

---

## EXAMPLES

Example 1 - ACCT only (code_ACCT=1, code_EFFI=blank)
"For health and safety SOPs, although AI would be easier, it would be me that would be culpable if something went wrong."
Why: "culpable" = liability framing

Example 2 - EFFI only (code_ACCT=blank, code_EFFI=1)
"AI provides me an email draft in less than a minute. The AI could extrapolate better in less time than me."
Why: "less than a minute", "less time" = time-saving

Example 3 - BOTH (code_ACCT=1, code_EFFI=1)
"I need to understand the research so I know what I'm saying is accurate. But AI helps me find the best approach when I'm not sure how to handle a complex matter."
Why: "know what I'm saying is accurate" = accountability; "helps me" = efficiency

Example 4 - NEITHER (code_ACCT=blank, code_EFFI=blank, code_OTHER="authenticity")
"The way AI writes doesn't sound like me at all. When others send AI-written messages, it feels like that person isn't trying to genuinely connect."
Why: authenticity/relational framing, not accountability or efficiency
